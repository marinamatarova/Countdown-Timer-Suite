# ⏱️ Countdown Timer Suite — , один таймер

Коллекция из семи независимых реализаций **таймера обратного отсчёта** с расширенными возможностями.

## Возможности (общие для всех версий)

- Установка времени в формате **ЧЧ:ММ:СС** или с помощью отдельных полей
- Кнопки **Старт**, **Пауза**, **Сброс**
- Звуковой сигнал по окончании (через системный beep или аудио)
- Визуальный прогресс‑бар (или кольцевая диаграмма)
- Готовые пресеты: **1 мин**, **5 мин**, **10 мин**, **15 мин**
- Отображение оставшегося времени крупным шрифтом
- Поддержка темной/светлой темы (в GUI и веб)

## Языки и интерфейсы

| Язык      | Интерфейс          | Особенности                                                                 |
|-----------|--------------------|-----------------------------------------------------------------------------|
| Python    | Tkinter (GUI)      | Звук через `winsound` или `pygame`, прогресс‑бар, цветовая индикация        |
| JavaScript| Веб (HTML5/CSS)    | Notification API, выбор времени через input, звук через Web Audio           |
| Go        | CLI (progress bar) | Своя анимация, системный beep, интерактивный ввод времени                   |
| Rust      | CLI + `crossterm`  | Цветной вывод, звук через `rodio` (или beep), обработка клавиш              |
| Java      | Swing              | Звук через `Toolkit.getDefaultToolkit().beep()`, таймер `javax.swing.Timer` |
| C#        | Windows Forms      | Асинхронный таймер, звук `Console.Beep` или `System.Media.SoundPlayer`      |
| PHP       | Веб (серверный)    | Сохранение пресетов в сессии, отправка звукового сигнала через JS на клиенте|
